# Syntaxwear - Tênis e Sneakers Online

![Syntaxwear Logo](./images/logo/logo.svg)

Bem-vindo ao repositório da **Syntaxwear**, uma landing page moderna e elegante desenvolvida para um e-commerce especializado em tênis e sneakers exclusivos. O projeto foca em uma experiência de usuário fluida, com design responsivo e uma estética urbana.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores práticas de desenvolvimento web front-end:

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização avançada utilizando:
    - **Variáveis CSS**: Para uma gestão centralizada de cores e fontes.
    - **Flexbox & CSS Grid**: Para layouts complexos e responsivos.
    - **Media Queries**: Garantindo que o site seja visualmente atraente em qualquer dispositivo (Desktop, Tablet e Mobile).
- **Google Fonts**: Utilização das fontes *Ubuntu* e *Inter* para uma tipografia moderna e legível.

## 🎨 Design e Estética

O projeto segue uma paleta de cores sofisticada, com destaque para o roxo (`#6329a2`), que confere uma identidade vibrante e tecnológica à marca. 

### Características principais:
- **Header Responsivo**: Inclui um menu hambúrguer funcional para dispositivos móveis sem o uso de JavaScript (utilizando a técnica de `checkbox hack`).
- **Seção Hero**: Destaque para o modelo principal "Krypton One" com chamadas para ação (CTAs) claras.
- **Categorias Dinâmicas**: Cards intuitivos para navegação rápida entre os estilos Casual, Esporte, Moderno e Futurista.
- **Grid de Produtos**: Um layout em grade que destaca produtos e modelos, proporcionando uma vitrine atraente.
- **Rodapé Completo**: Com formulário de inscrição em newsletter, links de navegação e integração com redes sociais.

## 📂 Estrutura do Projeto

A organização dos arquivos segue uma lógica modular para facilitar a manutenção:

```text
├── index.html          # Arquivo principal da página
├── css/                # Pasta contendo os módulos de estilo
│   ├── base.css        # Estilos globais e componentes básicos
│   ├── variables.css   # Definição de cores e fontes
│   ├── reset.css       # Reset de estilos padrão do navegador
│   ├── header.css      # Estilização do cabeçalho e navegação
│   ├── hero.css        # Estilização da seção de destaque
│   ├── product-category.css # Estilos para a seção de categorias
│   ├── product-grid.css # Estilos para a grade de produtos
│   └── footer.css      # Estilização do rodapé
└── images/             # Ativos visuais (logos, ícones e fotos)
```

## 🛠️ Como Executar

Para visualizar o projeto localmente, basta seguir os passos abaixo:

1. Clone o repositório para sua máquina local.
2. Navegue até a pasta do projeto.
3. Abra o arquivo `index.html` em qualquer navegador moderno.

Não é necessário nenhum servidor ou ferramenta de build, pois o projeto utiliza tecnologias web nativas.
